
# EmpowerHer - Final Full Stack Project
Created by Paul Mungai

Folders:
- empowerher-backend
- empowerher-frontend

Quick start:
1. Backend:
   - cd empowerher-backend
   - copy .env.example to .env and set MONGO_URI and JWT_SECRET
   - npm install
   - npm run seed
   - npm run dev

2. Frontend:
   - cd empowerher-frontend
   - npm install
   - create .env with VITE_API_URL=http://localhost:5000
   - npm run dev
